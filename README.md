Spring Boot Application with Kotlin, Java + Lombok + Groovy
===========================================================
Spring Boot application with Kotlin, Java + Lombok, Groovy and make them interoperable.

### Examples

* Integrate with Java 8 and Lombok
* Mix Java and Kotlin compile
* Groovy support
* DSL Examples
